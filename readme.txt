Fusion-C 1.3 is a free C library with which you can program software and games for MSX computers under MSX-DOS.
You must use Fusion-C with SDCC 4.0.0 compiler (higher versions have not been tested)
Library is providing functions to work with every aspects of MSX computers (MSX1, MSX2, MSX2+, MSX TURBO-C) easily. 
The package is ready to use with MacOS or Windows, once the different components have been installed. 

The Fusion-C Library can be supplemented by the book "FUSION-C COMPLETE JOURNEY" which you can buy on Amazon :
