How to Update from previous versions  to Fusion-C 1.3

All versions are using the same folder structure.

If you made changes inside the Fusion-C files, or Fusion-C folder save them
in another place.


- Rename The old "Working Folder" to "Working Folder Old" or copy it in a safe place.
- copy the new "workingFolder" where you want to use it.
- Put your sources files at the root of 'WorkingFolder'

Considering the compilation scripts changed, check the manual, chapter "Manual installation for windows" or "Manual installation for MacOs"
and check the graphic icon woith an exclamation point (!). This icon indicates the steps of the installation you must do for the update procerss.
